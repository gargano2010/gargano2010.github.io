Folder structure of the downloaded archive

In the main folder called "neatly" you will find the following:

neatly-template

  1. template assets
    • css contains all the stylesheets
    • img contains all the images used in the template
    • js contains all the javascript files
  2. index.html home page
  3. portfolio.html portfolio archive page
  4. case-study.html project presentation / case study page
  5. blog-classic.html classic blog archive with sidebar
  6. blog-grid.html masonry / grid blog layout
  7. blog-post.html blog post page with sidebar
  8. about.html about sample page
  9. contact.html contact page with working contact form
  10. page.html sample static page layout
  11. 404.html error 404 page
  12. send.php the script for the contact form

neatly-documentation

  1. _ documentation assets
  2. index.html documentation page
